Null Engine Model Format Version 1

Format

All values are in big endian

8 bit integer: version - 2 for current version1
32 bit integer: vertex count - The number of vertices
vertex count Vertices: vertices - The vertices
vertex count Texture Coorinates: texture coordinates - The texture coordinates
vertex count Normals: normals - The normals
32 bit integer: index count - The number of indices
index count 32 bit integers: indices - The indices

Vertex

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value

Texture Coordinate

Single precision float: x - The x value
Single precision float: y - The y value

Normal

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value
