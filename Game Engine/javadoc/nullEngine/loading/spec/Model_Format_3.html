Null Engine Model Format Version 3

Definitions

  • Level of detail - This is one of the levels used for distance simplifying where the first (index 0) is the most detailed
  • Total index count - The total of all the index counts

Changes From Version 2

  • Data for the levels of detail is now interleaved

Format

All values are in big endian

8 bit integer: version - 3 for current version1
8 bit integer: lod count - The number of levels of detail
lod count 32 bit integers: index count - The number of indices in each level
total index count 32 bit integers: indices - The indices to be used for the model
32 bit integer: vertex count - The total number of vertices across all levels of detail
vertex count Vertices: vertices - The vertices for all levels of detail
vertex count Texture Coorinates: texture coordinates - The texture coordinates for all levels of detail
vertex count Normals: normals - The normals for all levels of detail

Vertex

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value

Texture Coordinate

Single precision float: x - The x value
Single precision float: y - The y value

Normal

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value
