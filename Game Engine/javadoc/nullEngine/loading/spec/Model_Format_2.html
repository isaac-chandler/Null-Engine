Null Engine Model Format Version 2

Definitions

  • Level of detail - This is one of the levels used for distance simplifying where the first (index 0) is the most detailed

Changes From Version 1

  • Multiple levels of detail are now supported

Format

All values are in big endian

8 bit integer: version - 2 for current version1
8 bit integer: lod count - The number of levels of detail
lod count Levels of detail: lods - The levels of detail

Level of Detail

32 bit integer: vertex count - The number of vertices in this level of detail
vertex count Vertices: vertices - The vertices this level of detail
vertex count Texture Coorinates: texture coordinates - The texture coordinates for this level of detail
vertex count Normals: normals - The normals for this level of detail
32 bit integer: index count - The number of indices in this level of detail
index count 32 bit integers: indices - The indices for this level of detail

Vertex

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value

Texture Coordinate

Single precision float: x - The x value
Single precision float: y - The y value

Normal

Single precision float: x - The x value
Single precision float: y - The y value
Single precision float: z - The z value
